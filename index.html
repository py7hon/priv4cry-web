priv4cry.my.id

Selamat datang di priv4cry.my.id, kami memberikan sebuah service frontend privacy untuk menjaga privasi kalian dalam menjelajah internet.

Beberapa list service yang kami sediakan:

Social Media:
- YouTube: Piped | Music: Hyperpipe
- Pixiv: Freexiv
- Tiktok: ProxiTok

Kami menggunakan Cloudflare untuk DNS dan Tunnel dari localhost, dan untuk penyedia internet kami menggunakan AS141073(Selfhost) dan AS47311 untuk Gateway IIX

Kalian bisa melihat Network Usage di Selfhost Network dan Gateway IIX DC

Kalian bisa juga request dan berdiskusi tentang service ini di #priv4cry:matrix.iqbalrifai.eu.org

Priv4cry adalah project dari Muhammad Iqbal Rifai
